# wordTTY

A text-based client for the popular Wordle game. 

usage: `python3 wordle.py [--no-unicode]`  

alternatively, you can play it remotely:  
`telnet ssh.xereeto.co.uk 7777 `  
or  
`ssh wordle@ssh.xereeto.co.uk -p 9999`
